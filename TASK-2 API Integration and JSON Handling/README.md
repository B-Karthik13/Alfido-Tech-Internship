# Weather Information Finder using Python API

## Project Overview

This project fetches real-time weather data using the OpenWeatherMap API.

## Features

- API Integration using Requests
- JSON Parsing
- City Search
- Weather Filtering
- Error Handling
- Beginner Friendly

## Technologies Used

- Python
- Requests Library
- OpenWeatherMap API

## Installation

```bash
pip install requests
```

## Run the Project

```bash
python main.py
```

## Example Output

```text
City: Bengaluru, IN
Temperature: 29.34°C
Humidity: 57%
Weather: overcast clouds
Wind Speed: 4.93 m/s
```
